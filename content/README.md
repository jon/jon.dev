# Site content

The public writing and project indexes are defined in `site.ts`.

- Add a writing entry to the `writing` array. Its `slug` becomes
  `/writing/<slug>`.
- Add a project entry to the `projects` array.
- Commit and push the change to publish through the site’s Git workflow.

Keep dates in `YYYY-MM-DD` form and write a separate human-readable
`displayDate`.
